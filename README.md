### Installation
```bash
$ [sudo] pip install mac-app-frontmost
```

```bash
$ [sudo] npm i -g mac-app-frontmost
```

#### Examples
```bash
$ app-frontmost --bundleid
com.apple.Terminal

$ app-frontmost --name
Terminal

$ app-frontmost --path
/Applications/Utilities/Terminal.app

$ app-frontmost --pid
304
```
